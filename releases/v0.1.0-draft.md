## Cách tạo GitHub Release (command)

Sau khi 12 commit đã push, nhóm trưởng (Bình An) chạy:

```bash
# Từ root repo, trên default branch
git tag -a v0.1.0-draft -m "Pre-feedback package for instructor review (2026-06-06)"
git push origin v0.1.0-draft

# Sau đó trên GitHub:
# 1. Releases → Draft a new release
# 2. Chọn tag: v0.1.0-draft
# 3. Title: "v0.1.0-draft · Pre-feedback package for instructor review"
# 4. Description: paste nội dung file này (releases/v0.1.0-draft.md)
# 5. Đánh dấu "pre-release"  ← quan trọng — chưa phải S1 submission chính thức
# 6. Publish
```
